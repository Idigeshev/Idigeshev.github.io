# BI/Data Engineer

## О себе

Инженер данных с опытом работы в области Business Intelligence и Data Engineering. Занимаюсь разработкой BI-решений с 2022 года, а с 2024 года специализируюсь на Data Engineering. Обладаю комплексными навыками в построении аналитических систем, от сбора и обработки данных до создания интерактивных дашбордов и отчетов.

## Технологический стек

### Базы данных и хранилища
- **SQL базы данных:** PostgreSQL, MSSQL
- **Аналитические БД:** ClickHouse
- **Работа с различными источниками:** API, CSV, Excel, JSON

### Языки программирования и инструменты разработки
- **SQL** — написание и оптимизация сложных запросов
- **Python** — обработка и трансформация данных
- **GitHub Copilot** — ускорение разработки с помощью AI

### ETL/ELT и оркестрация
- **Apache Airflow** — построение и управление пайплайнами данных
- **dbt** — трансформация данных и моделирование

### BI и визуализация
- **Power BI** — создание интерактивных дашбордов
- **Datalens**
- небольшой опыт с Superset

## Основные компетенции

### Разработка ETL/ELT пайплайнов
- Проектирование и реализация процессов извлечения, трансформации и загрузки данных
- Автоматизация процессов обработки данных
- Обеспечение надежности и мониторинга пайплайнов

### Моделирование данных
- Проектирование Data Warehouse
- Создание эффективных схем данных 
- Построение витрин данных для аналитики (звёздочка, снежинка)

### BI и аналитика
- Разработка дашбордов и интерактивных отчетов
- Визуализация данных для принятия бизнес-решений
- Создание аналитических инструментов для различных отделов
- Ad-hoc аналитика — быстрое создание отчетов по запросу бизнеса
- Исследовательский анализ данных с построением детализированных таблиц метрик

### Оптимизация и производительность
- Оптимизация SQL-запросов для ускорения выполнения отчетов
- Анализ и улучшение производительности ETL-процессов
- Настройка индексов и partition для эффективной работы с большими объемами данных

### Автоматизация
- Автоматизация рутинных процессов обработки данных
- Разработка скриптов для регулярных задач
- Настройка автоматического обновления отчетов

## Области применения

- Построение end-to-end аналитических решений
- Интеграция данных из различных источников
- Создание централизованных хранилищ данных
- Разработка инструментов для бизнес-аналитики
- Оптимизация существующих процессов работы с данными
