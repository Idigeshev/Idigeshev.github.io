Дашборд план-факт

Заменили еженедельный ручной сбор данных из четырёх источников (3 инстанции IIKO + Google Sheets) на автоматический Python-скрипт и ежедневно обновляемый дашборд в Power BI.

Power BI Python MSSQL ETL
Открыть демо
Задачи
  • Видеть в ежедневной динамике отклонение факта от плана в разрезе до филиала
  • Контролировать показатели выручки и чека по дням, неделям, месяцам
  • Иметь единую картину по всем филиалам без ручной сборки данных
  • Автоматически обновлять дашборд ежедневно вместо ручной еженедельной выгрузки
Было → Стало
Было
  • Данные собирались вручную из 4 источников (3×IIKO + Google Sheets)
  • На сбор уходило несколько часов
  • Частота анализа только еженедельно
  • Высокий риск ошибок при ручном переносе данных
Стало
  • Python-скрипт собирает данные автоматически за 10 минут
  • Запускается ежедневно по расписанию
  • Дашборд обновляется каждый день без участия человека
  • Исключён ручной перенос — ошибки устранены
